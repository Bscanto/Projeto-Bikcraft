
# 🚴 Bikcraft

Projeto de website moderno e responsivo para uma marca fictícia de bicicletas elétricas premium chamada **Bikcraft**.

🔗 Deploy: https://projeto-bikcraft-omega.vercel.app/  
📦 Repositório: https://github.com/Bscanto/Projeto-Bikcraft

---

## 📖 Sobre o Projeto

O **Bikcraft** é um projeto front-end desenvolvido com foco em:

- Design moderno e responsivo
- Estruturação semântica em HTML
- Estilização avançada com CSS
- Interações utilizando JavaScript
- Organização de componentes e páginas
- Experiência do usuário (UX/UI)

O site apresenta produtos, planos, diferenciais da marca e informações institucionais de maneira visualmente elegante e intuitiva.

## 🚀 Como Executar o Projeto

### Clone o repositório

```bash
git clone https://github.com/Bscanto/Projeto-Bikcraft.git
```

### Acesse a pasta do projeto

```bash
cd Projeto-Bikcraft
```

### Execute o projeto

Basta abrir o arquivo `index.html` no navegador.

Ou utilize uma extensão como:

- Live Server (VS Code)
